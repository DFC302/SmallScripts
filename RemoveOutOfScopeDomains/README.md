# Remove Out Of Scope Domains

## Usage
```
usage: removeOutOfScopeDomains.py [-h] [--noscope NOSCOPE] [--scope SCOPE]

optional arguments:
  -h, --help            show this help message and exit
  --noscope NOSCOPE, -ns NOSCOPE
                        Specify file that contains out of scope domains.
  --scope SCOPE, -s SCOPE
                        Specify file that contains scoped domains.
```
